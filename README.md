## Getting Started 
To get a local copy up and running, follow these simple example steps.

### Prerequisites

Make sure you have Node.js  running on your local machine.

### Setup

~~~bash
$ git clone https://github.com/Godswilly/bookstore.git
$ cd bookstore
~~~

Install modules:

```
yarn install
```

### Usage

Start server with:

```
    yarn start or npm start
```
